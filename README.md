# Group Enhancer for Bases

Group Enhancer for Bases adds extra functionality to grouped views in [Obsidian Bases](https://obsidian.md/bases).

<img src="screenshot.gif" width="400" />

## Features

- Collapse and expand grouped sections
- Collapse all / Expand all actions
- Optional toolbar for grouped views
- Optional record counts beside group names
- Remembered collapse state between sessions

## Installation with BRAT

1. Install **BRAT** from Obsidian Community Plugins.
2. Open **Settings → BRAT → Add Beta Plugin**.
3. Add this repository:
   ```
   https://github.com/TfTHacker/group-enhancer-for-bases
   ```
4. Enable **Group Enhancer for Bases** in **Settings → Community Plugins**.

## Settings

### Toolbar
- Show feature toolbar
- Choose button display style: icon only, text only, or icon and text

### Collapsing Groups
- Apply collapsible groups by default
- Remember collapse state

### Count by Group
- Show counts by default

## Commands

- Collapse all groups in current Bases view
- Expand all groups in current Bases view

## Usage

1. Open a Bases view grouped by a property.
2. Enable the features you want in plugin settings.
3. If collapsible groups are enabled, click a group header to collapse or expand it.
4. Use the toolbar or command palette for Collapse all / Expand all.

## Notes

- This plugin is intended for grouped Bases views.
- Behavior is controlled through global plugin settings.
- The plugin has been tested on both small and large grouped datasets.
